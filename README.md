# AETHER Band Engine

AI-powered music generation engine with multi-agent pipeline architecture.

## Features

- Multi-agent pipeline for end-to-end music production
- Genre-aware composition and arrangement
- MIDI and audio rendering
- Professional mixing and mastering

## Installation

```bash
pip install aether-band-engine
```

## Quick Start

```bash
aether create "upbeat synthwave track" --genre synthwave
```
